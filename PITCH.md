# Vektor — AI 求职全流程助手

> 不只是简历编辑器，而是从「看到职位」到「投出简历」的完整 AI 工作流。

---

## 一句话定位

**Vektor 是一个 AI 驱动的求职工具，覆盖简历编辑、JD 智能匹配、Cover Letter 生成、简历评审、多语言翻译，并通过浏览器插件实现「边看职位边优化」的无缝体验。**

---

## 两大特色功能

### 特色一：浏览器插件 — 不离开招聘网站就能完成求职全流程

传统流程：看到职位 → 复制 JD → 打开简历工具 → 粘贴 → 分析 → 回到招聘网站……来回切换，效率极低。

**Vektor 的做法：你根本不需要离开招聘网站。**

Chrome 插件覆盖 **LinkedIn / Indeed / JobsDB / CTgoodjobs / Boss 直聘** 五大主流平台。打开任何一个职位详情页，插件自动识别站点、抓取完整 JD（职位名、公司、地点、岗位描述），页面右下角出现浮动卡片：

- **一键分析** — 当场看到匹配度评分，知道自己够不够格
- **一键保存** — 后台自动运行完整流水线：JD 分析 → 生成 Cover Letter → 简历定制建议，全部在后台完成
- **同步回主应用** — 所有分析结果、生成的求职信、简历修改建议，一键应用到简历，不需要手动抄

插件内置看板式求职追踪（Pending → Ready → Applied），每个投过的职位都有记录，不再靠 Excel 表格管理求职进度。

**它把「刷招聘网站」和「优化简历」这两件原本割裂的事，合成了一个动作。**

### 特色二：专为香港地区求职设计的三语切换

在香港求职有个独特的痛点：**同一个人可能需要简体中文、繁体中文、英文三个版本的简历。**

面试内地背景的公司用简体，投本地公司用繁体，外企用英文——三份简历要同步维护，改了一处其他两份都要跟着改。

Vektor 做了三件事来解决这个问题：

1. **简体 ↔ 繁体：一键精确转换**
   不是简单的字符替换。基于 opencc-js，正确处理「软件/軟件」「信息/資訊」「网络/網絡」这类用词差异。切换后简历内容整体转换，不需要逐字改。

2. **中文 ↔ 英文：AI 智能翻译**
   一键切到 English，Gemini AI 翻译全部简历内容。关键细节：
   - 公司名、学校名自动使用官方英文名（「腾讯」→「Tencent」，「香港大学」→「The University of Hong Kong」）
   - 技术术语、框架名原样保留（React、Docker、AWS 不会被翻译）
   - 职位名、学位名使用目标语言的惯用表达

3. **名称快照：切回来还是你原来的中文名**
   切到英文前，系统自动快照保存你手动输入的所有中文公司名、学校名。切回中文时，直接恢复快照内容，不依赖 AI 反向翻译——避免「Tencent」被翻回「腾讯控股有限公司」这种问题。

**一份简历维护三个语言版本，改一处同步所有。**

## 完整功能概览

### AI 能力矩阵

| 功能 | 做什么 | 怎么用 |
|---|---|---|
| JD 匹配分析 | 对比简历和目标职位，给出匹配度评分、缺失技能、ATS 关键词、逐条修改建议 | 粘贴 JD 或插件自动抓取 |
| 简历评审 | 四维评分（内容/影响力/完整性/清晰度）+ 优势总结 + 改进建议 + 逐章节反馈 | 一键启动 |
| 逐条 AI 优化 | 对话式打磨每一条工作描述，支持「量化数据」「突出成果」「STAR 方法」等快捷指令 | 点击 bullet 旁的 AI 按钮 |
| Cover Letter 生成 | 基于简历 + JD 生成求职信，三种风格（正式/友好/简洁），中英双语 | 一键生成，可在线编辑 |
| 智能翻译 | 简历内容中英互译，简繁互转 | 顶栏切换语言 |
| 简历解析 | 上传 PDF/Word，AI 自动提取信息填充所有字段 | 导入文件 |

### 编辑器功能

- **8 个板块**：个人信息、职业摘要、工作经历、教育背景、技能、项目、证书、语言
- **拖拽排序**：板块间拖拽调整顺序，板块内拖拽调整条目顺序
- **板块控制**：显示/隐藏任意板块，自定义简历结构
- **多简历管理**：同时维护多份简历，侧边栏快速切换
- **四套模板**：Modern / Classic / Minimal / Creative
- **主题定制**：8 个预设色 + 自定义颜色 + 4 种字体
- **实时预览**：A4 比例所见即所得
- **导入导出**：PDF / Word / JSON 导入，PDF / JSON 导出

### 浏览器插件

- **五站覆盖**：LinkedIn / Indeed / JobsDB / CTgoodjobs / Boss 直聘
- **自动抓取**：打开职位页自动提取 JD，无需手动复制
- **后台流水线**：保存职位后自动分析 + 生成 Cover Letter + 简历建议
- **结果同步**：分析结果一键应用到主应用的简历中
- **求职看板**：Pending → Ready → Applied 状态追踪
- **SPA 适配**：自动监听页面跳转，翻页不丢失

## 技术亮点

| 层级 | 技术 |
|---|---|
| 前端框架 | Vue 3 + TypeScript + Composition API |
| 状态管理 | Pinia（响应式 + localStorage 持久化） |
| 样式 | TailwindCSS 4 |
| AI 引擎 | Google Gemini API (gemini-3-flash-preview) |
| 中文转换 | opencc-js（简繁精确转换） |
| 构建 | Vite 7 |
| 浏览器插件 | Chrome Extension Manifest V3 + esbuild |
| 数据存储 | 纯本地 localStorage，零后端依赖 |

**极简架构**：生产依赖仅 5 个包，无 UI 组件库，纯前端完整运行，数据全部存在本地，无隐私顾虑。
